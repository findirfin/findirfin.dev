Your Task#

write a php program Using as many of these as you can

  • Functions
  • While Loops
  • For Loops
  • Input/Output
  • Arrays

Examples#

1. Functions

<?php
function greet($name) {
  return "Hello, " . $name . "!";
}

echo greet("Alice"); // Output: Hello, Alice!
?>

2. While Loops

<?php
$i = 1;
while ($i <= 5) {
  echo $i . " ";
  $i++;
}
// Output: 1 2 3 4 5
?>

3. For Loops

<?php
for ($i = 1; $i <= 5; $i++) {
  echo $i . " ";
}
// Output: 1 2 3 4 5
?>

4. Input/Output

<?php
// Output
echo "Enter your name: ";

// Input (from command line, requires running PHP from the terminal)
$name = readline();

echo "Hello, " . $name . "!";
?>

5. Arrays

<?php
// Indexed array
$colors = array("red", "green", "blue");
echo $colors[0]; // Output: red

// Associative array
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);
echo $ages["Peter"]; // Output: 35
?>

6. Conditionals

<?php
$age = 20;

if ($age >= 18) {
  echo "You are an adult.";
} else {
  echo "You are a minor.";
}
// Output: You are an adult.
?>
