To install Ubuntu on a Chromebook using chrx, follow these steps:

Preparation

1. Enable Developer Mode on your Chromebook. This will erase all local data, so back up any important files first[3].

2. Update your Chromebook's firmware if necessary, especially for Bay Trail, Braswell, and Apollo Lake models[4].

Installation Process

The installation using chrx is a two-phase process:

Phase 1: Allocating Space

1. Boot into Chrome OS and open the Terminal by pressing Ctrl + Alt + T[1].

2. Type `shell` and press Enter to open a shell[1].

3. Run the following command to download and execute chrx[4]:

```
curl https://chrx.org/ | sudo tar xzfC - /usr/local && chrx
```

4. Follow the on-screen instructions to allocate storage space for Ubuntu[4].

5. The system will reboot after this phase.

Phase 2: Installing Ubuntu

1. After rebooting, open the Terminal again and run the chrx command with your desired options[4]. For Ubuntu, use:

```
chrx -d ubuntu -r latest
```

You can replace `latest` with a specific version number if desired.

2. The installation will begin. This process may take some time, so be patient[1].

Post-Installation

Once the installation is complete, your Chromebook will reboot. You can then:

- Press Ctrl + L at startup to boot into Ubuntu[1].
- Press Ctrl + D to boot back into Chrome OS[1].

Additional Notes

- You can customize your installation further by using additional options with the chrx command. For example, to install a specific environment or architecture[4].
- If you encounter any issues, especially with Bay Trail Chromebooks, you may need to update your firmware using a script like the one mentioned in the search results[2].

Remember that installing Ubuntu will require significant space on your Chromebook, so ensure you have enough storage available before proceeding[3].
